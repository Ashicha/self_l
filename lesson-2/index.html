Challenge 1: Filter Cheap Flights Easy

Problem: Given an array of flights, write a function that returns only the flights with a price under a given budget AND that still have seats available (seatsLeft > 0).

📋 Requirements:
• Input: an array of flight objects, and a budget number
• Output: a new array with only flights where price < budget AND seatsLeft > 0
• If no flights match, return an empty array []

Test cases:

// Test data (same flights from your server): const flights = [ { flightNumber: "TG601", price: 12500, seatsLeft: 23 }, { flightNumber: "TG603", price: 18900, seatsLeft: 5 }, { flightNumber: "JL708", price: 15200, seatsLeft: 0 }, // sold out! { flightNumber: "TG659", price: 28500, seatsLeft: 8 }, { flightNumber: "SQ978", price: 6500, seatsLeft: 45 }, ]; findCheapFlights(flights, 15000) → ["TG601", "SQ978"] // TG601 (12500 < 15000, 23 seats ✅), SQ978 (6500 < 15000, 45 seats ✅) // JL708 excluded: 0 seats even though price is under budget findCheapFlights(flights, 5000) → [] // Nothing under 5000 findCheapFlights(flights, 50000) → ["TG601", "TG603", "TG659", "SQ978"] // All except JL708 (sold out)

Your code: Complete the function below:

💡 Hint:
1. Create an empty array for results: const result = [];
2. Loop through flights with for...of
3. Inside the loop, check TWO conditions with && (AND)
4. If both conditions pass, result.push(flight.flightNumber)
5. Return the result array
// Solution 1: Loop (clear and easy to read) function findCheapFlights(flights, budget) { const result = []; for (const flight of flights) { if (flight.price < budget && flight.seatsLeft > 0) { result.push(flight.flightNumber); } } return result; } // Solution 2: .filter() + .map() (shorter, more "developer" style) function findCheapFlights(flights, budget) { return flights .filter(f => f.price < budget && f.seatsLeft > 0) .map(f => f.flightNumber); }

Challenge 2: Find the Best Deal Easy

Problem: Write a function that finds the cheapest available flight between two airports. If no flights are available, return null.

📋 Requirements:
• Input: array of flights, origin string, dest string
• Filter by: origin, dest, AND seatsLeft > 0
• Return: the full flight object with the lowest price, or null
// Test cases: findBestDeal(flights, "BKK", "NRT") → { flightNumber: "TG601", price: 12500, ... } // cheapest available BKK→NRT findBestDeal(flights, "NRT", "SIN") → null // no flights on this route
💡 Hint:
1. Start with let cheapest = null;
2. Loop through all flights
3. Check if origin matches AND dest matches AND seatsLeft > 0
4. If match: compare with current cheapest — if cheaper (or cheapest is null), update it
5. After loop, return cheapest
function findBestDeal(flights, origin, dest) { let cheapest = null; for (const flight of flights) { if (flight.origin === origin && flight.dest === dest && flight.seatsLeft > 0) { if (cheapest === null || flight.price < cheapest.price) { cheapest = flight; } } } return cheapest; }

Challenge 3: Seat Availability Summary Medium

Problem: Write a function that takes an array of seat objects and returns a summary object counting how many seats are available, held, and booked.

📋 Requirements:
• Input: array of seat objects, each with { seat: "1A", status: "available" }
• Output: an object { available: X, held: X, booked: X }
• Possible statuses: "available", "held", "booked"
// Test: const seats = [ { seat: "1A", status: "available" }, { seat: "1B", status: "booked" }, { seat: "1C", status: "available" }, { seat: "2A", status: "held" }, { seat: "2B", status: "available" }, { seat: "2C", status: "booked" }, ]; seatSummary(seats) → { available: 3, held: 1, booked: 2 }
💡 Hint:
1. Create a counter object: const summary = { available: 0, held: 0, booked: 0 };
2. Loop through seats
3. Check each seat's status with if/else if
4. Increment the matching counter: summary.available++ (++ means "add 1")
5. Return the summary object
function seatSummary(seats) { const summary = { available: 0, held: 0, booked: 0 }; for (const seat of seats) { if (seat.status === "available") { summary.available++; } else if (seat.status === "held") { summary.held++; } else if (seat.status === "booked") { summary.booked++; } } return summary; }

Challenge 4: Apply Promo Code Medium

Problem: Your flowchart has "POST /api/promos/validate". Write a function that applies a promo code to a price. Each promo has different rules.

📋 Requirements:
• Input: original price (number), promo code (string)
• Valid promos: "SAVE10" = 10% off, "FLAT500" = ฿500 off, "SUMMER20" = 20% off
• If promo code is invalid, return { valid: false, message: "Invalid promo code" }
• If valid, return { valid: true, originalPrice, discount, finalPrice }
• Final price should never be negative (minimum ฿0)
// Test cases: applyPromo(12500, "SAVE10") → { valid: true, originalPrice: 12500, discount: 1250, finalPrice: 11250 } applyPromo(12500, "FLAT500") → { valid: true, originalPrice: 12500, discount: 500, finalPrice: 12000 } applyPromo(12500, "SUMMER20") → { valid: true, originalPrice: 12500, discount: 2500, finalPrice: 10000 } applyPromo(12500, "FAKE99") → { valid: false, message: "Invalid promo code" } applyPromo(300, "FLAT500") → { valid: true, originalPrice: 300, discount: 300, finalPrice: 0 } // ↑ 300 - 500 = -200, but minimum is 0, so finalPrice = 0, discount = 300
💡 Hint:
1. Use if/else if to check the code string
2. For "SAVE10": discount = price * 0.10 (10%)
3. For "FLAT500": discount = 500 (fixed amount)
4. Use Math.max(0, price - discount) to prevent negative prices
5. The actual discount = price - finalPrice (in case it was capped at 0)
function applyPromo(price, code) { let discount = 0; if (code === "SAVE10") { discount = price * 0.10; } else if (code === "FLAT500") { discount = 500; } else if (code === "SUMMER20") { discount = price * 0.20; } else { return { valid: false, message: "Invalid promo code" }; } const finalPrice = Math.max(0, price - discount); const actualDiscount = price - finalPrice; return { valid: true, originalPrice: price, discount: actualDiscount, finalPrice: finalPrice }; }

Challenge 5: Build an API Route Handler Medium

Problem: This simulates writing a real Express route. Given a req object (the request), write the logic that would go inside app.get("/api/flights", ...). Return the correct response object.

📋 Requirements:
• Input: req object with req.query (like { origin: "BKK", dest: "NRT" })
• If origin is missing → return { status: 400, body: { error: "Missing origin" } }
• If dest is missing → return { status: 400, body: { error: "Missing dest" } }
• If both present → filter flights and return { status: 200, body: { count: X, flights: [...] } }
💡 Hint:
1. const origin = req.query.origin; (or destructure: const { origin, dest } = req.query;)
2. Check: if (!origin) → return { status: 400, body: { error: "Missing origin" } }
3. Filter: flights.filter(f => f.origin === origin && f.dest === dest)
4. Return: { status: 200, body: { count: results.length, flights: results } }
function handleFlightSearch(req, flights) { const { origin, dest } = req.query; if (!origin) { return { status: 400, body: { error: "Missing origin" } }; } if (!dest) { return { status: 400, body: { error: "Missing dest" } }; } const results = flights.filter( f => f.origin === origin && f.dest === dest ); return { status: 200, body: { count: results.length, flights: results } }; }

Challenge 6: Complete Booking Price Calculator Hard

Problem: Build a function that calculates the total booking price with add-ons and promo. This combines EVERYTHING you've learned!

📋 Requirements:
• Input: a booking object with flight price, passengers count, add-ons array, and optional promo code
• Add-on prices: "baggage" = ฿800, "meal" = ฿350, "insurance" = ฿500
• Add-ons are PER PASSENGER
• Promo applies to the TOTAL (after add-ons). Same promos as Challenge 4.
• Return: { basePrice, addOnsTotal, subtotal, discount, totalPrice, breakdown }
// Test: calculateBooking({ flightPrice: 12500, passengers: 2, addOns: ["baggage", "meal"], promoCode: "SAVE10" }) → { basePrice: 25000, // 12500 × 2 passengers addOnsTotal: 2300, // (800 + 350) × 2 passengers subtotal: 27300, // 25000 + 2300 discount: 2730, // 10% of 27300 totalPrice: 24570, // 27300 - 2730 }
💡 Hint (step by step):
1. basePrice = flightPrice × passengers
2. Create an add-on price map: const addOnPrices = { baggage: 800, meal: 350, insurance: 500 };
3. Loop through booking.addOns, sum up prices using the map
4. Multiply add-on sum by passengers
5. subtotal = basePrice + addOnsTotal
6. Apply promo to subtotal (reuse your Challenge 4 logic!)
7. Return the result object
function calculateBooking(booking) { const addOnPrices = { baggage: 800, meal: 350, insurance: 500 }; // Step 1: Base price const basePrice = booking.flightPrice * booking.passengers; // Step 2: Calculate add-ons per person let addOnPerPerson = 0; for (const addon of booking.addOns) { if (addOnPrices[addon]) { addOnPerPerson += addOnPrices[addon]; } } const addOnsTotal = addOnPerPerson * booking.passengers; // Step 3: Subtotal const subtotal = basePrice + addOnsTotal; // Step 4: Apply promo let discount = 0; if (booking.promoCode === "SAVE10") { discount = subtotal * 0.10; } else if (booking.promoCode === "FLAT500") { discount = 500; } else if (booking.promoCode === "SUMMER20") { discount = subtotal * 0.20; } const totalPrice = Math.max(0, subtotal - discount); discount = subtotal - totalPrice; return { basePrice, addOnsTotal, subtotal, discount, totalPrice }; }
